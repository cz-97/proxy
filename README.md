# clash客户端自定义脚本，加工你的订阅。
- 适合含有低倍率路线的代理商
- 删除代理商提供的规则，加入[clash-rules](https://github.com/Loyalsoldier/clash-rules)、[blackmatrix7](https://github.com/blackmatrix7/ios_rule_script)和本人的自定义规则
- 对于能用base64做icon的客户端，如[Clash Party](https://github.com/mihomo-party-org/mihomo-party),可以用[base64图标版脚本](./config_base64_icon.js)，其他客户端用[链接图标版脚本](./config.js)。 更改[链接图标版脚本](./config.js)后, 需要执行`bun svg2b64.js`以同步更改[base64图标版脚本](./config_base64_icon.js)
